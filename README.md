# WeatherBot
## _Телеграм бот, показывающий погоду_

Что умеет бот:

- Показывать погоду
- Запоминать пользователя
- Очищать информацию о пользователе
- Показывать анекдоты
- Делать рассылку сообщений всем, кто когда-либо регистрировался в боте (и не удалял себя из БД._.)
